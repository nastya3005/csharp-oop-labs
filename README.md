# Лабораторные работы по ООП и паттернам проектирования

## О проекте
Коллекция лабораторных работ, выполненных в рамках университетского курса по ООП и паттернам проектирования. Реализованы на C# с акцентом на чистую архитектуру, SOLID, тестируемость.

---

## Содержание

| Лаб | Тема | Технологии |
|-----|------|------------|
| 1 | Симулятор транспортного маршрута | ООП, модульные тесты |
| 2 | Система распределения сообщений | GRASP, SOLID, структурные паттерны, mocking |
| 3 | Симулятор сражений фэнтези-существ | Порождающие и структурные паттерны |
| 4 | Файловый менеджер (консоль) | Поведенческие паттерны, парсер команд |
| 5 | Система банкомата (Web API) | Гексагональная архитектура, ASP.NET Core |

---

## Кратко о каждой

**Лаб 1.** Симулятор движения поездов по маршруту. Участки: обычные пути, силовые пути, станции. Реализована физика движения, 8 тестовых сценариев.

**Лаб 2.** Корпоративная система доставки сообщений. Форматтеры (консоль/файл), архиваторы, системы оповещений, фильтрация по важности. Mocking в тестах.

**Лаб 3.** Боевая система. Существа с уникальными способностями, модификаторы (магический щит, мастерство атаки), заклинания. Расширяемость без нарушения OCP.

**Лаб 4.** Консольный файловый менеджер. Команды: connect, disconnect, tree, file (show/move/copy/delete/rename). Unix-синтаксис путей, ленивая загрузка данных.

**Лаб 5.** Web API банкомата. Гексагональная архитектура, сессии (пользовательская/админская), счета, баланс, пополнение/снятие, история операций. In-memory репозитории, MSDI.
