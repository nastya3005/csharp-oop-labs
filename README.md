# csharp-oop-labs
Лабораторные работы по ООП и паттернам проектирования на C#. Симулятор транспортных маршрутов, система распределения сообщений, боевой симулятор, файловый менеджер, банкомат на ASP.NET Core. Использованы SOLID, порождающие и структурные паттерны, гексагональная архитектура, модульные тесты.
